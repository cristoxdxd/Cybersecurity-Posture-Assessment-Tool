# Cybersecurity-Posture-Assessment-Tool

A comprehensive web application for evaluating organizational cybersecurity posture based on **NIST CSF 2.0**, **CIS Controls v8**, and **NIST SP 800-53 Rev. 5** frameworks.

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/cristoxdxd/Cybersecurity-Posture-Assessment-Tool.git
cd Cybersecurity-Posture-Assessment-Tool
```

### 2. Create a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```bash
python -m pip install -e .
```

## Usage

### Start the Application

```bash
python run.py
```

The application will be available at `http://localhost:5000`
